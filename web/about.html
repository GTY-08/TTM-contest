요청부터 완료까지 한 화면 흐름으로 이어집니다

틈틈은 요청자와 작업자 역할을 따로 만들지 않습니다. 한 계정으로 맡기고, 지원하고, 진행 상황을 확인합니다.

1. 요청 탭
요청 생성

필요한 심부름을 바로 등록

내용, 위치, 예상 시간, 보상, 직접 입력 태그만 간결하게 넣습니다.

유연한 시간 입력사진과 메모직접 입력 태그
2. 매칭 방식
빠

대기형 빠른 매칭

조건이 맞는 작업자가 바로 수락합니다. 짧고 즉시성이 높은 요청에 맞습니다.

요청 공개알림
작업자 수락매칭
일

일반 지원형 매칭

서로 금액을 제시하고 마지막 제시 금액에 양쪽이 동의해야 선정됩니다.

지원자 제안금액
양쪽 동의선정
3. 찾기 탭
찾기

가까운 작업을 빠르게 비교

거리, 시간, 태그와 사진을 보고 지원할 작업을 고릅니다.

지원함 표시태그 검색추천 작업
4. 홈 탭
진행

매칭된 작업과 지원한 작업

지갑

시연용 가상 지갑 흐름

신뢰

본인확인과 안전 기능

5. 진행 화면

채팅

요청자와 작업자가 진행 중 필요한 내용을 바로 주고받습니다.

위치 확인

진행 중 필요한 범위에서 위치 흐름을 확인합니다.

인증 사진

요청자가 사진을 보고 승인하거나 반려할 수 있고, 반려 시 다시 촬영합니다.

6. 계정과 인증

웹/앱 같은 계정

웹에서 가입한 계정은 앱 로그인에도 그대로 사용됩니다.

본인확인 방식

은행인증서 본인인증과 PASS 본인인증을 선택할 수 있도록 준비되어 있습니다.
